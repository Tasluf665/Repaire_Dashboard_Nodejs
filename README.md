# Repair - Electronic Device Repair Service App

Welcome to the documentation for the Repair app, an Android application built with React Native Expo. Repair is an e-commerce platform that allows users to hire technicians to repair their electronic devices. This project comprises three core components: the Android app (developed using React Native Expo), the backend server (powered by Node.js), and the admin panel (created with React). MongoDB serves as the primary database.

## Admin Panel Features

1. **Role-Based Authentication**:
   - Secure role-based authentication ensures that only authorized users can access the admin panel.

2. **Customer Orders Handling**:
   - Efficient management of customer orders, enabling administrators to track and manage repair requests.

3. **Agent and Technician Handling**:
   - Streamlined handling of agents and technicians, facilitating their onboarding and management.

4. **Interactive Admin Panel**:
   - An intuitive and interactive admin panel offers administrators a seamless experience.

5. **Login with Password or Google Login**:
   - Multiple login options are available for administrators, enhancing convenience and security.

## Getting Started

To explore the Repair Admin Panel, follow these steps:

1. **Clone the Repository**: Clone this GitHub repository to your local machine.

2. **Install Dependencies**: Install the necessary dependencies by running the following commands:

   ```bash
   npm install
3. **Run the Admin Panel**: Start the admin panel by running:

   ```bash
   npm start
   
  This will launch the admin panel locally, allowing you to explore its features.
